Sobre o Projeto

Este sistema foi desenvolvido para realizar o cadastro e análise de vinhos artesanais da Vinharia Agnello.

Utilizando JavaScript básico, o usuário pode inserir informações como nome do vinho, tipo, safra e quantidade em estoque.

Funcionalidades

  • Cadastro de vinhos
  • Exibição de dados no console
  • Interação com o usuário via prompt e alert
